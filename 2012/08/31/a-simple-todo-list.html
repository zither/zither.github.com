PHP生成嵌套列表

31 Aug 2012 - By MiniMee

花了点时间把Minilist Demo的代码重新写了一遍，这次换了一个角度来思考问题。既然ul li列表是成对出现的，那么就可以采用正则匹配来替换生成列表，完整代码如下：

<?php
/*
 * 使用PHP通过正则匹配生成嵌套列表。
 * 
 * @author minimee <ilorn.mc@gmail.com>
 *
 */

class lists {

    /**
     * 默认缩进为4个空格。
     *
     * @var int
     */    
    private $tab = 4;

    /**
     * 默认允许最大嵌套数。
     *
     * @var int
     */
    private $level = 4;

    /**
     * Todolist
     *
     * @var string
     */ 
    private $list = NULL;

    /**
     * 初始化类
     *
     * @return void
     */
    public function __construct($list)
    {
        $this->list = $list;
    }

    /**
     * 获取处理后的Todolist
     *
     * @return string
     */
    public function get_list()
    {
        // 替换根列表
        $this->parent();

        // 替换子列表
        $this->child();

        // 最终列表
        $this->list = '<ul>'.$this->list.'</ul>';

        return $this->list;
    }

    /**
     * 匹配替换根列表
     *
     * @return void
     */
    protected function parent()
    {
        $mregex = '/^\S.+\n((\s{'.$this->tab.',}.+\n)*)/im';

        $replacement = '<li>$0</li>';

        $this->list = preg_replace($mregex,$replacement,$this->list);
    }

    /**
     * 循环匹配替换子列表
     *
     * @return void
     */
    protected function child()
    {
        // 在允许的最大嵌套数内进行循环替换子列表
        for($i = 1; $i < $this->level; $i++){

            // 当前列表缩进宽度
            $tab1 = $this->tab*$i;

            // 当前列表的子列表缩进宽度
            $tab2 = $tab1+1;

            $regex = '/^\s{'.$tab1.'}\S.+\n((\s{'.$tab2.',}\S.+\n)*)/im';

            $replacement = '<ul><li>$0</li></ul>';

            $this->list = preg_replace($regex,$replacement,$this->list);
        }
    }
}

// 读取数据内容
$contents = file_get_contents('mlist.txt');

$list = new lists($contents);

echo $list->get_list();

这次代码比之前的代码更加简单明了，已经完全满足我自己的需要了，可能在嵌套层数方面可以改进一下。
